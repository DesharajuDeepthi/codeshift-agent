# 17 — Version 2 Roadmap

Do not implement until V1 is complete.

## Candidate additions

1. SQLAlchemy 1.x to 2.x migration pack
2. Python 3.10 to 3.12 runtime pack
3. OSV vulnerability intelligence
4. deps.dev dependency graph
5. GitHub Actions run-history analysis
6. private repositories through a least-privilege GitHub App
7. organization policy packs
8. GitHub issue creation after human approval
9. multi-pack migration dependency planning
10. deployment to LangSmith Agent Server or another container platform

## Version 3 boundary

Autonomous code transformation, sandboxed test execution, repair loops, and pull-request creation belong to Version 3 and require additional security design and human approval gates.
